## 정확도 테스트 질문 (문서에 답이 있음)

### Q1. 단순 조회
**질문**: 한소희 담당자의 총 거래 건수는 몇 건인가요?
**정답**: 289건
**검증 방법**: `df[df['담당자'] == '한소희']` 행 수

### Q2. 집계 질문
**질문**: 제품E의 총 판매수량은 얼마인가요?
**정답**: 159,726개
**검증 방법**: `df[df['제품명'] == '제품E']['판매수량'].sum()`

### Q3. 순위 질문
**질문**: 매출액이 가장 높은 담당자는 누구인가요?
**정답**: 이영희 (총 매출액 약 84억)
**검증 방법**: `df.groupby('담당자')['매출액'].sum().idxmax()`

### Q4. 목록 질문
**질문**: 어떤 부서들이 있나요?
**정답**: 운영부, 영업부, 재무부, 인사부, 연구소, 품질관리부, 기획부, 고객지원부, 마케팅부, 개발부
**검증 방법**: `df['부서'].unique()`

### Q5. 특정 조건 질문
**질문**: 서울 지역의 판매 데이터가 있나요?
**정답**: 예, 있습니다 (서울이 지역 목록에 포함됨)
**검증 방법**: `'서울' in df['지역'].unique()`

## 할루시네이션 테스트 질문 (문서에 없는 내용)

### H1. 존재하지 않는 담당자
**질문**: 김영수 담당자의 매출 실적은 어떻게 되나요?
**정답 기대**: "김영수라는 담당자는 데이터에 없습니다" 또는 유사 응답
**검증**: 김영수는 데이터에 없음

### H2. 존재하지 않는 제품
**질문**: 제품Z의 판매현황을 알려주세요.
**정답 기대**: "제품Z는 데이터에 없습니다" 또는 유사 응답
**검증**: 제품Z는 데이터에 없음 (제품A~E, 서비스X~Z, 솔루션1~2만 있음)

### H3. 존재하지 않는 지역
**질문**: 제주도 지역의 매출은 얼마인가요?
**정답 기대**: "제주도 데이터는 없습니다" 또는 유사 응답
**검증**: 제주도는 지역 목록에 없음

### H4. 데이터에 없는 정보 질문
**질문**: 2024년 매출 데이터를 보여주세요.
**정답 기대**: "2024년 데이터는 없습니다" 또는 유사 응답
**검증**: 데이터는 2025년 10-11월만 있음
