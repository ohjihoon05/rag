# Excel Calamine 품질 테스트

## 개요

Excel calamine 엔진 적용 후 RAG 품질을 검증합니다.
- 파싱 성능 비교 (openpyxl vs calamine)
- 질문-응답 정확도 테스트
- 할루시네이션 체크

## 테스트 목표

1. **성능 테스트**: calamine 적용 후 파싱 속도 측정
2. **정확도 테스트**: 질문에 대해 문서 기반 정확한 답변 여부
3. **할루시네이션 테스트**: 문서에 없는 내용을 생성하는지 확인

## 테스트 방법

### 1. 성능 테스트
- 동일 Excel 파일 파싱 시간 비교
- Before: openpyxl (기존)
- After: calamine (변경 후)

### 2. 정확도 테스트
- Excel 파일 업로드 → Knowledge Base 생성
- 미리 준비한 질문 목록으로 쿼리
- 응답과 원본 문서 비교

### 3. 할루시네이션 테스트
- 문서에 없는 내용에 대한 질문
- "모르겠다" 또는 정확한 거부 응답 여부 확인
